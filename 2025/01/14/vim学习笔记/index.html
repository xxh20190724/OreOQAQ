vim文件管理

  1. 分割窗口：显示两个文件或一个文件不同地方，比较两个文件；

    • 分割 :split 新文件 :new
    • 跳转 ：<c-w> w
    • 窗口移动 ：<c-w> HJKL
    • 关闭 :close
    • 保留一个 ：only

  2. 窗口大小

    • 指定3行的窗口 :3split 文件
    • 扩大窗口3行 ：3<c-w> +
    • 缩小窗口3行 ：3<c-w> -
  • 垂直分割 :vsplit :vnew
  • 跳转 <c-w> hjkl/tb
  • 批量打开窗口
    vim -o file1 file2 file3
    批量存关文件 :wqall
  1. diff
    1. 命令行中 vimdiff file1~file2
    2. :edit file.txt

    :vertical diffsplit file2.txt
    3. :edit file.txt
    :vertical diffpatch file.txt 补丁不会用
    4. set noscrollbind 同步滚动
    5. 差异跳转
    ]c [c 支持记数
    6. 差异编辑 用于增加几行拷到其他 文件中
    本文 do “obtain
    其他 dp “put

查找替换

  1. 查找
    • :/

vim技巧

在一个窗口中，VIM 的小tab标签页命令：

  • :tabnew [++opt选项] ［＋cmd］ 文件 建立对指定文件新的tab
  • :tabc 关闭当前的tab
  • :tabo 关闭所有其他的tab
  • :tabs 查看所有打开的tab
  • :tabp 前一个
  • :tabn 后一个

分屏

  • :vsplit 这个命令太长，没人用的。大家都用 :vsp ，:sp filename来分屏。
  • 分屏的相关动作都是ctrl+w 开始的，然后再跟一个其他字母
  • ctrl+w = ：让左右上下各个分屏宽度，高度均等。
  • ctrl+w _(shift + -) :当前屏幕高度扩展到最大
  • ctrl+w |(shift + ) :当前屏幕宽度扩展到最大
  • ctrl+w c：关闭当前屏幕

竖向编辑

  • Ctrl+Shift+v
  • I输入 esc

括号内容编辑

  • di(删除括号()内容
  • da(删除括号和内容

自动对齐

  • 选中后=
